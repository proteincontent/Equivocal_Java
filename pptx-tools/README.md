# PPTX 生成工具

用于在不改动项目根目录依赖的情况下，生成《Equivocal Legal 开题答辩》PPT。

## 使用方法

```bash
cd pptx-tools
npm install
npm run build:proposal
```

输出文件默认生成在项目根目录：

- `Equivocal_Legal_开题答辩.pptx`

## Aurora 版本（Light Mode · 产品发布会风）

```bash
cd pptx-tools
npm install
npm run build:proposal:v2
```

输出文件：

- `Equivocal_Legal_Masterpiece_opening_v2.pptx`
